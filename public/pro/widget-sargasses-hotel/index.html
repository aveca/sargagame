Installation

Trois étapes, deux minutes, aucun développeur.

Si vous savez coller un bloc dans WordPress, Wix, Squarespace ou n'importe quel CMS, vous savez installer le widget.

Choisissez votre plage et le format

Sur le générateur, sélectionnez la plage la plus proche de votre établissement, la langue, le format (complet ou compact) et votre couleur. L'aperçu se met à jour en direct.

Copiez le code

Un seul bloc <iframe> à copier — celui affiché ci-dessus. Aucun compte, aucune inscription, aucun script à installer.

Collez-le dans votre page

Insérez le code dans un bloc HTML / « code embed » de votre site (page d'accueil, page « la plage », confirmation de réservation…). C'est en ligne. Le widget vit ensuite sa vie et se rafraîchit tout seul.
