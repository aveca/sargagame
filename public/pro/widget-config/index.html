Configurez votre widget

Réglez votre widget « plages en direct », générez le code, collez-le sur votre site. Le même satellite qui veille votre rivage, à votre marque : avec votre token Pro, le widget passe à vos couleurs — sans crédit Sargasses, mesuré au satellite, pas deviné.

Comment ça marche

  1. Collez le token Pro qu'on vous a envoyé par email (active la marque blanche).
  2. Choisissez votre plage, vos couleurs, votre format dans l'aperçu.
  3. Copiez le code et collez-le dans votre site (une simple iframe, zéro maintenance).
Sans token : widget gratuit, avec un discret crédit « par Le Veilleur » — la confiance que vous donnez au client.

Conditions en direct par Sargasses Martinique

Plusieurs plages ?

Chaque widget affiche une plage. Pour en afficher plusieurs (ex. plusieurs plages autour de votre établissement), répétez simplement le bloc avec une autre plage sélectionnée — votre token Pro reste le même.

Besoin d'un token, ou d'aide ?

Le token Pro vous est envoyé par email à la souscription. Pour le renouveler, l'étendre à une autre plage, ou toute question d'intégration, écrivez-nous à alerte@sargasses-martinique.com.
