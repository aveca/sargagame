Démarrage

Aucune clé requise pour les données publiques. Une simple requête GET renvoie du JSON :

# Martinique — état + prévision de toutes les plages
curl -s https://sargasses-martinique.com/api/copernicus/sargassum.json

# Guadeloupe
curl -s https://sargasses-guadeloupe.com/api/copernicus/sargassum.json

CORS ouvert en lecture : utilisable depuis un navigateur, un script, ou un tableau de bord. Soyez raisonnable sur la fréquence (la donnée ne bouge que 4×/jour — un cache d'1 h suffit largement).

GET État & prévision

GET /api/copernicus/sargassum.json

L'état du jour de chaque point de mesure + la prévision 7 jours. Extrait :

{
  "source": "erddap-live",
  "updatedAt": "2026-06-21T04:13:25Z",
  "dataAgeMinutes": 142,
  "levels": [
    { "id":"grande-anse", "afai":0.05, "status":"clean",
      "confidence":70, "h2s":{ "score":0, "level":"low" } }
  ],
  "weekly": {
    "grande-anse": { "forecast": [
      { "day":"Auj.", "date":"2026-06-21", "status":"clean",
        "afai":0.05, "confidence":70, "regime":"calm" }
    ] }
  }
}

Champs

ChampTypeSens
statusstringclean · moderate · avoid — le verdict baignade.
afainumberIndice AFAI (Algae Floating Area Index) mesuré au large, 0 → ~1. La matière première du statut.
confidence0–100Fiabilité de la lecture du jour (fraîcheur satellite + concordance des sources).
h2sobjectIndice de risque H2S (gaz de décomposition) dérivé de la masse accumulée — level = low/moderate/high. Pas une mesure de gaz in situ.
weekly[id].forecast[]array7 entrées (J0→J6) : date, status, afai, confidence, regime.
updatedAt / dataAgeMinutesiso / intFraîcheur. Affichez toujours l'âge : ne servez jamais une donnée muette.

GET Fiabilité publiée

GET /api/copernicus/track-record.json

Notre fiabilité, auditée et publiée — le contraire d'une boîte noire. Taux de justesse du statut, erreur AFAI moyenne, ventilés par horizon et par régime de saison (calme / transition).

{
  // exemple de structure — valeurs réelles servies à l'endpoint, datées
  "overall": { "statusHitRatePct":…, "afaiMAE":… },
  "byRegime": { "calm":{…}, "transition":{…} },
  "byHorizon": {…}, "sampleSize":…, "window":…
}

Règle d'or : aucun chiffre inventé. Le taux est publié daté et par régime — tous régimes confondus, jamais présenté comme une justesse de haute saison. Un global masque que les alertes de saison calme sont en faible confiance — on l'étiquette. Mesuré au satellite, pas deviné — et quand on se trompe, on l'écrit. Voir la page lisible : /fiabilite/.
