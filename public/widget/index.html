Comment l'intégrer

Copiez ce code HTML et collez-le sur votre site. Remplacez anse-mitan par le slug de la plage souhaitée.

  src="https://sargasses-martinique.com/widget/embed/anse-mitan/"
  width="100%" height="90"
  frameborder="0"
  title="Sargasses Anse Mitan - état en temps réel"
